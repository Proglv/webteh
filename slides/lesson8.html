isset(mainīgais)

  • Funkcija isset() atgriež vērtību „patiess”, ja šāds mainīgais ir izveidots un „nepatiess”, ja tas nav izveidots.
  • Funkcija darbojas ne tikai ar sesijas mainīgajiem, bet ar jebkuru mainīgo. 
if (!isset($_SESSION["skaits"])){
      $_SESSION["skaits"] = 1;
}else{
      $_SESSION["skaits"]++;
}

Apache piekļuves kontrole

  • Direktīva "Allow"
    • Ar šīs direktīvas palīdzību var atļaut piekļuvi kādam resursam

Apache piekļuves kontrole

  • Direktīva "Deny" -direktīva "Deny" aizliedz piekļuvi kādam resursam.
  • Direktīva "Order"- šī direktīva nosaka kādā kārtībā notiks un apstrāde.

Apache piekļuves kontrole

  • Direktīva
    • Ar šīs direktīvas palīdzību var noteikt pieejas tiesības failu sistēmas direktoriju koka ietvaros.
    • "ceļš" norāda uz kuru direktoriju un tās apakšdirektorijiem tas attieksies.
    • Apache pēc noklusēšanas ļauj caur URL piekļūt visiem failu sistēmas failiem. 

Order Allow,Deny
Allow from All

Apache piekļuves kontrole

  • Direktīva
    • Ar šīs direktīvas palīdzību var noteikt pieejas tiesības konkrētiem failiem vadoties pēc to nosaukuma (vai maskas) neatkarīgi no to atrašanās vietas failu sistēmā. 

Order deny, allow
Allow from all

Apache piekļuves kontrole

  • Direktīva
    • Ar šīs direktīvas palīdzību var noteikt pieejas tiesības kādai tīmekļa vietnes vietai vadoties pēc URL.
    • Šādas tiesības var darboties arī attiecībā uz dinamiski ģenerētām lapām. 

Order Deny,Allow
Deny from All
Allow from va.lv
