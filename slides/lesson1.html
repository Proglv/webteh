Tagi un to atribūti (I)

  • Tagi ir HTML elementi, kas norāda kā attēlot vai citādi apstrādāt dokumenta tekstu.
  • <tags...> Taga saturs </tags>
  • Tagus un tekstu kas atrodas starp atverošo un aizverošo tagu, sauc par taga saturu.
  • Atsevišķiem tagiem nav obligāti nepieciešams noslēdzošais tags, jo starp to sākuma un beigu tagu nav atļauti citi tagi vai teksts.
  • <br .../>

Tagi un to atribūti (II)

  • Pirmais vārds (vai burts/i) tagā ir arī tā nosaukums, kas parasti diezgan labi apraksta taga funkciju.
  • Nākamie vārdi tagā ir tā atribūti, kuriem ir konkrēta vērtības, kas norādītā pēc vienādības zīmes '='.
  • Piemērs:
<a href="http://www.va.lv">ViA</a>

Meta informācija

  • title tags - lapas nosaukums, kas parādās pārlūkprogrammas loga augšā.
  • meta tags ar name="keywords" - atslēgvārdi priekš Google u.c. meklētājiem
  • meta tags ar atribūtu charset - norāda faila kodējumu (UTF-8 ir unikods) 

<head>
<title>Lapas virsraksts</title>
<meta name="keywords" content="lekcija, teorija, html, meta" />
<meta charset="UTF-8">
</head>

Speciālie tagi

  • <link> - lai ielādētu ārēju CSS failu
  • <style> - lai izmantotu dokumentā iekļauto CSS
  • <script> - Lai izmantotu JavaScript 

<link rel="stylesheet" type="text/css" href="stili/stils.css" />

!DOCTYPE (I)

  • HTML dokumenta sākumā (1.rindā) jābūt doctype tagam kas norāda dokumenta tipu, lai pārlūkprogrammai būtu zināms kura specifikācija izmantota dokumenta veidošanā.
  • http://www.w3.org/QA/2002/04/valid-dtd-list.html

!DOCTYPE (II)

  • Strict - nav pieļaujami HTML elementi, kas nodrošina vizuālo noformējumu.
  • Transitional - iekļauti arī vizuālā noformējuma elementi, kas atļauj jau HTML dokumentā noteikt burtu, līniju izmēru un krāsu, kā arī citas īpašības.
  • Frameset - ja vēlaties HTML dokumentā izmantot freimus.
  • <!DOCTYPE html> būs OK!

Saites uz citiem resursiem

  • Saites uz resursiem norādīt to adresii izmanto:
    • lai izveidotu saitas uz citiem dokumentiem,
    • lai iekļautu attēlus, kuri jāparāda HTML dokumentā.
  • Absolūtais ceļš norāda pilnu ceļu līdz interesējošai datnei. Parasti tas tiek izmantots, ja vēlaties norādīt saiti uz citā tīmekļa vietnē esošiem resursiem. <a href="/mape2/dok2.html">
  • Relatīvais ceļš nozīmē, ka tiek norātīts ceļš uz citu resursu attiecībā pret html dokumentu,kurā šī saite norādīta. <a href="dok2.html">
  • Vienas tīmekļa vietnes ietvaros ieteicams lietot tikai relatīvos ceļus.

    • Relatīvais ceļš

    • Direktorijas vai faila nosaukums
    • . → esošā direktorija
    • .. -> vecāku direktorija
    • / -> direktoriju atdalīšanai

    HTML attēlošana

    • HTML ievadītās līnijas pārneses (enter u.c.) netiek attēlotas pārlūkprogrammā.
    • Lai kārtotu teksta struktūru izmantojiet <p>, <br> u.c. tagus

    HTML attēlošana

    • Bloka līmeņa tagi - aizņem maksimālu platumu, attēlojas viens zem otra.
    • Rindas līmeņa tagi - aizņem minimālu platumu, attēlojas blakus viens otram.

    Bloka līmeņa tagi (I)

    • <h1>,<h2>,<h3>,<h4>,<h5>,<h6>
    • <p>
    • <ul>, <ol> un <li>
